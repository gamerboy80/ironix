Privacy


At Ironix, we value your privacy; but we need some essential data for offering our services, also you can help our services to be in good conditions by providing data. The data we get to provide our services is:
- Server info
- Channel info
- User info

We also optionally collect data to help our services, called "bug reports". The data collected when a command fails if you opted in is:
- Full command
- Time it was run
- Server info
- Channel info
- User info
- Error
